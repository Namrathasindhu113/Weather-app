# 🌦️ Weather App

A modern and responsive Weather Application that allows users to check real-time weather conditions of any city across the world.

---

## 🚀 Features

- 🌍 Search weather by city name  
- 🌡️ Real-time temperature updates  
- 💨 Displays wind speed and humidity  
- ⛅ Dynamic weather conditions & icons  
- 📱 Responsive design (works on mobile & desktop)  
- ⚡ Fast and user-friendly interface  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript  
- Weather API (e.g., OpenWeatherMap)

---

## 📂 Project Structure

```

Weather-app/
│── index.html
│── style.css
│── script.js
│── assets/

````

---

## ⚙️ How to Run Locally

1. Clone the repository  
```bash
git clone https://github.com/Namrathasindhu113/Weather-app.git
````

2. Navigate into the project folder

```bash
cd Weather-app
```

3. Open `index.html` in your browser

---

## 🔑 API Setup

1. Get your API key from OpenWeatherMap
2. Replace it inside `script.js`:

```js
const apiKey = "YOUR_API_KEY";
```
